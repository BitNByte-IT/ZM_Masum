# Images folder

Drop your photos here, then reference them in `src/data.js` as
`images/<filename>` (no leading slash).

Recommended files:
- portrait.jpg        — Z M Masum portrait (square, ~800×800)
- zsm.jpg             — ZSM Transport Agency (16:9)
- alishan.jpg         — Alishan Transport (16:9)
- sonarbangla.jpg     — Sonar Bangla Transport Agency (16:9)
- fleet-1.jpg, fleet-2.jpg, fleet-3.jpg  — gallery shots (4:3)

Any missing image automatically shows a styled placeholder slot,
so the site looks clean even before you add photos.
